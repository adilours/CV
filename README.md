# Adil Mhira - CV / Portfolio

Interactive CV and portfolio website featuring 3D animated avatars.

## Features

- **Dual Mode**: Normal mode and "Zero Bullshit" mode with different visual styles
- **3D Animations**: GLB models with Draco compression rendered via Three.js
- **Bilingual**: French and English content
- **Responsive**: Optimized for all screen sizes

## Tech Stack

- HTML5 / Tailwind CSS
- Three.js with GLTF/Draco loaders
- Vanilla JavaScript
- GitHub Pages hosting

## Live Site

[https://adilours.github.io/CV/](https://adilours.github.io/CV/)

## Local Development

Simply open `index.html` in a browser or serve with any static server:

```bash
npx serve .
```
